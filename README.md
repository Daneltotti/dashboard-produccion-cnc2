 – Comparativa Herramienta Actual vs Nueva

Este proyecto presenta un análisis completo del impacto económico y productivo al sustituir la herramienta **Actual** por una **Nueva** en un proceso CNC.  
El dashboard está desarrollado en **Power BI**, utilizando medidas DAX para evaluar scrap, costos, beneficios y eficiencia operativa.

---

Objetivo del Proyecto

El propósito del análisis es responder a tres preguntas clave:

¿La herramienta nueva reduce el scrap?
¿La herramienta nueva mejora el beneficio neto?  
¿Cuánto dinero se evita perder al cambiar de herramienta?

El dashboard permite visualizar estas respuestas de forma clara, comparativa y accionable.
